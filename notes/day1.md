1. What is VPC?
Ans:- A virtual private cloud (VPC) is a secure, isolated private cloud hosted within a public cloud.

2. What is VPC CIDR?
Ans:- Classless Inter-Domain Routing (CIDR) is an IP address allocation method that improves data routing efficiency on the internet.

3. Public vs Private Subnet?
Ans:- The subnet has a direct route to an internet gateway is called public subnet and The subnet does not have a direct route to an internet gateway is called private subnet.

4. Route table vs Internet Gateway?
Ans:- An Internet Gateway provides the connection point between your VPC and the internet, while Route Tables define the routing rules for traffic within your VPC, including directing internet-bound traffic to the Internet Gateway.

5. What is Terraform module?
Ans:- Terraform is an infrastructure as code tool that lets you build, change, and version infrastructure safely and efficiently.
